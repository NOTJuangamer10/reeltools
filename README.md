# 🎬 RGG

> Don't you ever spend hours watching reels and see videos with GitHub repositories
> or websites that are super useful, but you don't know when you'll actually use them?
> RGG is here to solve that. It's a local tool that saves video links, repository names,
> transcripts, and descriptions.

## 🚀 Live demo

🔗 **<https://notjuangamer10.github.io/reeltools/>**

## ✨ Features

- 📋 Paste any reel URL (Instagram, TikTok, YouTube…)
- 🏷️ Add a name/title to each saved video
- 📝 Keep a description of what the video is about
- 🎙️ Store the transcript of the video
- 💾 Everything is saved locally in your browser (localStorage) — no account needed
- 🗑️ Delete entries you no longer want

## 🛠️ Tech stack

| Layer    | Technology          |
|----------|---------------------|
| Markup   | HTML5               |
| Styles   | CSS3                |
| Logic    | Vanilla JavaScript  |

No frameworks, no build step, no dependencies.

## ▶️ Run locally

```bash
git clone https://github.com/notjuangamer10/reeltools.git
cd reeltools
# then just open index.html in your browser
```

Or simply download and double-click `index.html`.

## 📂 Project structure

```
reeltools/
├── index.html   # page structure
├── style.css    # styles (warm orange theme)
└── app.js       # logic: add / list / delete entries
```
